# Pulse

Pulse - это современный интернет-магазин спортивной одежды и атрибутики, разработанный с использованием фреймворка Django. Проект предлагает полный цикл покупок: от просмотра товаров до оформления заказа и отслеживания истории покупок. Цель проекта - предоставить пользователям удобную и интуитивно понятную платформу для покупки спортивной одежды онлайн.

## Основные функции

•   **Каталог товаров с фильтрацией:**  Удобный просмотр товаров с возможностью фильтрации по категориям и цене

•   **Детальная информация о товаре:**  Страницы с подробным описанием товараи и фотографиями

•   **Корзина покупок:**  Возможность добавления товаров в корзину, изменения количества, удаления товаров и просмотра общей стоимости заказа

•   **Система аутентификации:**  Регистрация новых пользователей, вход в систему для зарегистрированных пользователей, управление профилем (изменение личных данных, адресов доставки)

•   **Оформление заказов:**  Простой и удобный процесс оформления заказа

•   **История покупок:**  Просмотр истории заказов, отслеживание статуса текущих заказов

•   **Управление платежной информацией:**  Безопасное хранение платежных данных

•   **Административная панель:** Удобный интерфейс для администраторов для управления каталогом товаров, заказами, пользователями и другими аспектами работы магазина

## Технологический стек

•   **Backend:**

    •   Django 5.2 (Python) - основной фреймворк для разработки серверной части

•   **Frontend:**

    •   Tailwind CSS - CSS фреймворк для быстрой и удобной разработки пользовательского интерфейса

•   **База данных:**

    •   SQLite (в разработке) - для локальной разработки и тестирования

    •   PostgreSQL (в продакшене) - для надежного и масштабируемого хранения данных в продакшене

•   **Другое:**

    •   Git - система контроля версий

    •   GitVerse - платформа для хостинга репозитория кода
